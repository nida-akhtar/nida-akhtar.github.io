02 / Projects

Things I've built.

A selection of projects spanning retrieval systems, computer vision, deep learning, and applied ML. Each one was an excuse to learn something new — code and full details live on my GitHub.

RAG · LLMs
Constitution of Pakistan RAG
A retrieval-augmented generation system that answers questions strictly from the 1973 Constitution of Pakistan. Built with LangChain, Chroma DB, and locally-run models through Ollama (DeepSeek-R1 for generation, nomic-embed-text for embeddings). The goal was a system that grounds every answer in retrieved text rather than hallucinating.
Computer Vision
GViTOn — Virtual Glasses Try-On
A virtual try-on application for eyewear, built around facial landmark detection and image overlay. A fun project that taught me a lot about working with face geometry and real-time image manipulation.
LLMs · Tooling
LEXA — DeepSeek Code Companion
A Streamlit-based coding assistant powered by Ollama and LangChain. LEXA runs locally and helps with code explanations, debugging, and writing snippets — useful when you'd rather not send your code to a hosted API.
Search · IR
Lumina — Corpus Search
A search system over a document corpus, named after Lucene and "illumination." This project deepened my understanding of information retrieval — indexing, ranking, and the surprising amount of engineering behind a working search box.
Time Series · Transformers
Stockify — Stock Price Prediction
A Transformer-based architecture for next-day stock price prediction. The model itself was an experiment in applying attention-based architectures to financial time series, with the usual caveats about how hard markets are to actually predict.
Applied ML · Healthcare
Alternative Medicine Finder
A platform that helps users find alternative medications with similar active ingredients — designed to address the very real problem of drug shortages in Pakistani pharmacies. One of my projects that sits closest to my growing interest in machine learning for healthcare.
