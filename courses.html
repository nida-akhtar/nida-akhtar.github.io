What I've been studying.

I'm in my final year of BS Computer and Information Sciences at the Pakistan Institute of Engineering and Applied Sciences (PIEAS). Below is a selection of the courses that have shaped how I think about computing and machine learning.

AI, Machine Learning & Computer Vision

CIS-315
Introduction to Artificial Intelligence
CIS-452
Fundamentals of Machine Learning
CIS-428
Introduction to Computer Vision
CIS-322
Signal and Image Processing
CIS-422
Information Storage and Retrieval
How search engines work: indexing, ranking, and retrieval models. Directly informed my Lumina and RAG projects.
CIS-461
Data Mining
Pattern discovery in large datasets: the practical side of finding signal in noisy, real-world data.

Core Computer Science

CIS-318
Data Structures and Algorithms
The classics: useful in some way every single week of my degree.
CIS-221
Analysis of Algorithms
Complexity analysis, algorithm design strategies, and the discipline of asking how well something scales before celebrating it.
CIS-301
Operating Systems
Concurrency, memory management, and scheduling: the layer that makes everything above it possible.
CIS-401
Compiler Construction
Lexers, parsers, and code generation. Few courses change how you read code as much as this one does.
CIS-442
Parallel and Distributed Processing
Designing systems that run on many machines: the foundation for any serious work with large models or datasets.
CIS-317
Theory of Automata

Systems & Software Engineering

CIS-303
Database Systems
CIS-305
Internet Application Development
CIS-301
Software Engineering
CIS-304
Computer Security
CIS-202
Computer Organization and Architecture
How computers actually work at the hardware level: instructions, memory hierarchies, and pipelines.
CIS-210
Data Communication and Computer Networks
Protocols, network layers, and the surprisingly elegant systems that move data across the world.
CIS-421
STCIS — Practical IoT
Embedded systems and the Internet of Things: connecting computation to the physical world.

Mathematics

PAM-242
Linear Algebra
The language ML actually speaks. Worth revisiting often.
PAM-267
Probability and Statistics
The other language of machine learning, and the foundation for honest reasoning about uncertainty.
PAM-220
Calculus and Analytic Geometry
Derivatives, integrals, and the mathematics underneath every optimization-based learning algorithm.
PAM-255
Differential Equations
Modeling change: useful in ways that aren't always obvious until you need them.

Numerical Methods
Computational approaches to mathematical problems: and a healthy respect for floating-point arithmetic.
PAM-126
Physics-I
